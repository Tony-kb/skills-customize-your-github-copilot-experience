# 📘 Assignment: Python Data Visualization

## 🎯 Objective

Learn how to communicate data insights using Python charts and graphs. In this assignment, you will load data, create visualizations with matplotlib or plotly, and interpret patterns from the plotted results.

## 📝 Tasks

### 🛠️ Load and Prepare the Dataset

#### Description
Write code to read a CSV file and prepare it for charting by selecting relevant columns and checking for missing or invalid values.

#### Requirements
Completed program should:

- Load `sample-data.csv` using Python.
- Print the first 5 rows to confirm the data loaded correctly.
- Validate that numeric columns are treated as numbers.
- Handle missing values by either filling or removing incomplete rows.


### 🛠️ Create Two Different Charts

#### Description
Build at least two visualizations that represent the data in different ways and include clear chart labels.

#### Requirements
Completed program should:

- Create one line chart showing trends over time.
- Create one bar chart comparing values across categories.
- Add a title, x-axis label, and y-axis label to each chart.
- Display or save the charts from your Python program.


### 🛠️ Summarize Insights from the Visuals

#### Description
Use your charts to write a short interpretation of what the data shows.

#### Requirements
Completed program should:

- Write 3-5 sentences explaining key patterns or trends.
- Mention at least one peak or low point observed in a chart.
- Compare at least two categories based on chart results.
- Include one example run in a code block showing expected output behavior.

Example run:

```text
Loaded sample-data.csv
Rows loaded: 12
Created line chart: monthly_sales_trend.png
Created bar chart: region_comparison.png
Insight summary saved to insights.txt
```
